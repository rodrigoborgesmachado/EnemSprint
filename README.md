# EnemSprint

Treino para provas do ENEM usando a API publica do Sunsale.

## Requisitos
- Node.js 18+

## Como rodar
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Funcionalidades
- Listagem e busca de provas ENEM, IFTM e UFU
- Timer com duracao customizavel
- Navegacao por questoes e grade de acesso rapido
- Resultado completo com graficos e revisao
