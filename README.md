# EnemSprint

Treino para provas do ENEM usando a API publica do Sunsale.

## Requisitos
- Node.js 18+

## Como rodar
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Funcionalidades
- Listagem e busca de provas ENEM
- Timer com duracao customizavel
- Navegacao por questoes e grade de acesso rapido
- Resultado completo com graficos e revisao

## Screenshots (exemplo)
1. Inicie o app com `npm run dev`.
2. Abra `http://localhost:5173`.
3. Capture as telas:
   - Home com lista de provas
   - Durante uma prova (timer + questao)
   - Resultado com graficos e revisao
